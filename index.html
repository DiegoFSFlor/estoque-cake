SCE by vitorpc

Sistema de Controle de Estoque

O sistema tem enfoque no controle de estoque de uma loja que possui necessidade de gerir a quantidade de produtos e suas saídas, além de registrar os funcionários que efetuaram a operação e exibir relatórios de entradas/saídas. O SCE é especifico para a finalidade que se propõe, não abrangendo transações financeiras. O sistema está sendo desenvolvido em linguagem PHP com o framework CakePHP utilizando como base de armazenamento de dados o SGBD (Sistema de Gerenciamento de Banco de Dados) MySQL.

Dependencies

Pré-requisitos: Para instalar o SCE é necessário um Servidor HTTP, Apache com o mod_rewrite habilitado é preferido, o PHP instalado e configurado, e o SGBD MySQL.

Install

1. Descompactação do arquivo: Após descompactar o arquivo colocar a pasta no diretório root do servidor HTTP. 2. Configurações da Base de dados: É necessário alterar o arquivo SCE_ROOT/app/config/database.php, com as configurações da base de dados. 3. Criação de Tabelas e dados iniciais no banco: Executar o script sql SCE_ROOT/app/config/schema/dumpfile.sql. 4. Permissões: Dar permissao total de escrita e leitura à pasta SCE_ROOT/app/tmp. 5. Nome da empresa: Alterar “LOJA X” para o nome desejado no arquivo SCE_ROOT/app/views/layouts/default.ctp na linha 30. 6. Testando a aplicação: Para acessar basta ir no navegador e digitar http://localhost/SCE_ROOT. O usuário inicial é admin e a senha admin.
